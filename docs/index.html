Welcome to LinchFin’s documentation!¶

LinchFin is a financial engine library to build your own portfolio. This library is on development.

Installation¶

git clone https://github.com/shephexd/linchfin
python setup py install

How to use¶

In [1]: import linchfin as lf
In [2]: lf
  alpha: float
  animated: [True | False]
  antialiased or aa: [True | False]
  ...snipw
