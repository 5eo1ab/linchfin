Welcome to LinchFin's documentation!
====================================
LinchFin is a financial engine library to build your own portfolio.
This library is on development.

Installation
============
.. sourcecode:: bash

   git clone https://github.com/shephexd/linchfin
   python setup py install

How to use
==========
